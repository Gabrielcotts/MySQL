# Estudos e Exercícios de MySQL

Este repositório reúne meus estudos, exercícios e projetos desenvolvidos durante o aprendizado de **MySQL**, com o objetivo de fortalecer os conhecimentos em modelagem de banco de dados, linguagem SQL e boas práticas de desenvolvimento.

Ao longo das atividades, pratico desde a criação de bancos de dados e tabelas até consultas mais elaboradas, aplicando conceitos fundamentais utilizados no desenvolvimento de sistemas.

## Conteúdos estudados

* Criação de bancos de dados (`CREATE DATABASE`);
* Criação e alteração de tabelas (`CREATE`, `ALTER` e `DROP`);
* Tipos de dados;
* Chaves Primárias (Primary Key);
* Chaves Estrangeiras (Foreign Key);
* Relacionamentos entre tabelas;
* Modelagem de dados (MER e DER);
* Normalização (1FN, 2FN e 3FN);
* Inserção, atualização e exclusão de dados (`INSERT`, `UPDATE` e `DELETE`);
* Consultas utilizando `SELECT`;
* Filtros com `WHERE`;
* Ordenação com `ORDER BY`;
* Operadores (`AND`, `OR`, `NOT`, `BETWEEN`, `IN`, `LIKE`);
* Funções de agregação (`COUNT`, `SUM`, `AVG`, `MIN` e `MAX`);
* Agrupamentos com `GROUP BY` e `HAVING`;
* Joins (`INNER JOIN`, `LEFT JOIN` e `RIGHT JOIN`);
* Scripts para criação e população de bancos de dados.

## Objetivo do repositório

Este repositório faz parte da minha jornada de formação em Engenharia de Software e tem como finalidade registrar minha evolução no estudo de bancos de dados relacionais. Os exercícios e projetos aqui desenvolvidos servem como prática dos conceitos aprendidos e como base para aplicações futuras utilizando Java, Spring Boot e outras tecnologias do ecossistema back-end.

## Tecnologias utilizadas

* MySQL
* MySQL Workbench
* SQL
